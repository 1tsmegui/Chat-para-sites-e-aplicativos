<h1>Documentação do Projeto: Chat Online em Python</h1>

<h3>Este projeto implementa um sistema de chat online que pode ser integrado a sites e aplicativos. Ele utiliza a biblioteca flet para criar uma interface de usuário interativa e funcional.</h3>

<h3>Funcionalidades Implementadas</h2>
<h2>Iniciar Chat:</h2>

<h3>Um botão "Iniciar chat" é exibido inicialmente.
Ao clicar neste botão, abre-se um pop-up onde o usuário pode inserir seu nome para entrar no chat.</h3>
<h2>Entrar no Chat:</h2>

<h3>Após inserir o nome e clicar em "Entrar no chat", o pop-up inicial é fechado.
O título inicial e o botão "Iniciar chat" desaparecem.
É exibido um chat vazio com uma mensagem informando que o usuário entrou no chat.</h3>
<h2>Enviar Mensagem:</h2>

<h3>No chat, há um campo de texto onde o usuário pode digitar mensagens.
Ao pressionar "Enviar", a mensagem é exibida no chat junto com o nome do usuário.</h3>
<h2>Componentes Utilizados</h2>
<h3>Biblioteca: flet é utilizada para criar a interface de usuário.
Componentes UI: Text, TextField, ElevatedButton, Row, Column, AlertDialog são utilizados para construir a interface e interações do usuário.</h3>

<h2>Como Utilizar</h2>
<h3>Execução: Execute o código Python para iniciar a aplicação.
Interatividade: Clique em "Iniciar chat" para abrir o pop-up inicial, insira seu nome e clique em "Entrar no chat" para começar a enviar mensagens.
Envio de Mensagens: Digite uma mensagem no campo de texto abaixo do chat e clique em "Enviar" para que ela seja exibida no chat.</h3>
<h2>Conclusão</h2>
<h3>Este projeto proporciona uma interface simples e funcional para um chat online, permitindo integração com websites e aplicativos que desejem oferecer interação em tempo real entre usuários.</h3>
